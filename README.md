# Messaging App

A full-stack real-time messaging application that allows users to communicate seamlessly. This project demonstrates modern web development practices including user authentication, real-time communication, and responsive design.

## Features

- **User Authentication**: Secure registration and login system with JWT - tokens are stored in local storage
- **Real-time Messaging**: Send and receive messages with other users
- **User Profiles**: Customize your profile with personal information and preferences
- **Profile Picutres**: Profile picuture can be easily removed or updated, and it will be saved in cloudinary.
- **Message History**: View conversation history with other users
- **Theming**: Toggle between dark and light theme with one click - your prefernece will be saved in Local Storage.
  -- **Guest Login**: Be able to sign in as a guest without creating your account.

## Technologies Used

- **Frontend**: [React/ Vite]
- **Backend**: Node.js with [Express]
- **Database**: Prisma with [Postgres]
- **Authentication**: Passport.js with [JWT]
- **Form Validation**: Express Validator
- **Styling**: [styled-components]
- **CDN**: Cloundinary
- **Backend Deployment Platform + Postgres Server**: Railway
- **Frontend Deployement Platform**: Netlify
